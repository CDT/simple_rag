## ✨ 使用方法

界面已自动切换为中文，无需任何额外配置。

启动应用后，所有文本将以中文显示：

```bash
cd frontend
npm run dev
```

打开 `http://localhost:5173` 即可看到完整的中文界面。

## 🌐 未来扩展

如需支持多语言切换，可以：

1. 创建语言文件：`locales/zh-CN.json` 和 `locales/en-US.json`
2. 使用 Vue I18n 库
3. 添加语言切换器到设置页面

当前实现为固定中文界面，简洁高效。
